# Student Performance Predictor

Hey there! This is a simple Python app I built to predict student final marks based on study hours, attendance, and previous marks. It's my fun project for Fundamental AI & ML.

## Quick Start

1. Make sure you have Python and these libraries: `pip install scikit-learn numpy tkinter` (tkinter usually comes built-in).
2. Run `python project1.py`.
3. Enter your inputs in the GUI and hit Predict!

## How It Works

- Uses a basic Linear Regression model trained on sample data.
- GUI built with Tkinter – super straightforward.
- Handles bad inputs gracefully.
