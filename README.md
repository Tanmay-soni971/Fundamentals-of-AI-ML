# 📊 Student Performance Predictor

A simple yet effective **Machine Learning project** that predicts a student's final marks based on:
- 📚 Study Hours  
- 🏫 Attendance  
- 📝 Previous Marks  

Built as part of a **Fundamentals of AI & ML** project, this app combines a basic ML model with a user-friendly GUI.

---

## ✨ Features

- 🎯 Predicts final marks using **Linear Regression**
- 🖥️ Easy-to-use **Tkinter GUI**
- ⚡ Instant predictions with a click
- 🛡️ Handles invalid inputs smoothly
- 🧠 Beginner-friendly ML implementation

---

## 🛠️ Tech Stack

- Python 🐍  
- scikit-learn 🤖  
- NumPy 🔢  
- Tkinter 🖼️  

---

## 🚀 Getting Started

### 1. Clone the Repository
git clone  
cd student-performance-predictor  

### 2. Install Dependencies
pip install scikit-learn numpy  

*(Tkinter usually comes pre-installed with Python)*

### 3. Run the Application
python project1.py  

---

## 🧪 How It Works

1. A **Linear Regression model** is trained on sample student data.
2. The model learns relationships between:
   - Study hours  
   - Attendance  
   - Previous marks  
3. The Tkinter GUI collects user input.
4. The model predicts and displays the final marks instantly.
